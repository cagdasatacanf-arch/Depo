## 🆘 QUICK FIXES

| Problem | Fix |
|---------|-----|
| GitHub Desktop won't show changes | Close/Reopen GitHub Desktop |
| Can't find "Show in Finder/Explorer" | Right-click Depo → Open options |
| Files won't commit | Add text in Summary box |
| Push fails | Click Push again, wait 5 sec |
| Can't see changes after adding files | Wait 5 seconds, GitHub Desktop auto-detects |
| Saved as .txt instead of .json | Rename: right-click → Rename → aapl.json |

## 💡 PRO TIPS

✅ Copy JSON exactly - no changes!
✅ Filenames lowercase (aapl.json not AAPL.json)
✅ Save before closing text editor
✅ GitHub Desktop watches folder automatically
✅ Push takes 3-5 seconds
✅ Verify on GitHub.com after pushing
